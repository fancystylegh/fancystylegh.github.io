CSS选择器！

一. 基本选择器

  1. 通用选择器，会将所有元素匹配并设置样式。

  2. 元素选择器

  3. id选择器

  4. 类选择器

也可以使用”元素.class”的形式，限定某种类型的元素。

  1. 属性选择器,中括号里是属性名
    [href]{

    color: red;

    }

    [type=”password”]{

    border: 1px solid red;

    }

    [href^=”http”]{

    color: red;

    }

    [href$=”.com”]{

    color: red;

    }

    [href*=”baidu”]{

    color: red;

    }

    [class~=”abc”]{

    font-size: 12px;

    }

    HTML5

    [lang|=”en”]{

    color: red;

    }

二.复合选择器–将不同的选择器进行组合形成新的特定匹配

  1. 分组选择器
    p,span,h1{

    color: red;

    }
    将多个选择器通过逗号分隔，同时设置一组样式。

  2. 后代选择器
    span p{

    color: red;

    }

  3. 子选择器 只能向下一级
    ul > li{

    border: 1px solid red;

    }

  4. 相邻兄弟选择器 匹配和第一个元素相邻的第二个元素
    p + b{

    color: red;

    }

  5. 普通兄弟选择器 匹配和第一个元素后面的所有的元素
    p ~ b{

    color: red;

    }

三. 伪元素选择器

伪选择器分为两种：第一种是下节伪类选择器（:），第二种是伪元素选择器(::)

1. ::first-line块级首行
::first-line{
    color: red;
}

2. ::first-letter块级首字母
::first-letter{
    color: red;
}

3. ::before文本前插入
a::before{
    content: "点击-";
}

4. ::after 文本后插入
a::after{
    content: "-请进"
}

四. 结构性伪类选择器

结构性伪类选择器能够根据元素在文档中的位置选择元素。这类元素都有一个前缀（:）

  1. 根元素选择器,基本不用，总是返回元素
    :root{

    border: 1px solid red;

    }

  2. 子元素选择器,
    ul > li:first-child{

    color: red;

    }
    选择第一个li

    ul > li:last-child{

    color: red;

    }
    选择最后一个li

    ul > li:only-child{

    color: red;

    }
    选择只有一个子元素的那个子元素

    div > p:only-of-type{

    color: red;

    }
    选择只有一个指定类型的子元素的那个子元素

  3. :nth-child(n)系列
    ul > li:nth-child(2){

    color: red;

    }
    选择子元素的第二个元素

    ul > li:nth-last-child(2){

    color: red;

    }
    选择子元素的倒数第二个元素

    div > p:nth-of-type(2){

    color: red;

    }
    选择特定子元素的第二个元素

    div > p:nth-last-of-type(2){

    color: red;

    }
    选择特定子元素的倒数第二个元素

五. UI伪类选择器

UI伪类选择器是根据元素的状态匹配元素

1. :enabled{
    border: 1px solid red;
}

选择启用状态的元素

2. :disabled{
    border: 1px solid red;
}

选择禁用状态的元素

3. :checked{
    display: none;
}

选择勾选的input元素

4. :default{
    display: none;
}

从一组类似的元素中选择默认元素。比如input被勾选的就是默认的。

5. :valid和:invalid

input:valid{
    border: 1px solid red;
}
input:invalid{
    border: 1px solid green;
}

输入验证合法与不合法显示时选择的元素

6. :required和:optional

input:required{
    border: 1px solid red;
}
input:optional{
    border: 1px solid green;
}

是否具有required属性选择元素

六. 动态伪类选择器–根据条件的改变匹配元素

1. :link和:visited
a:link{
    color: red;
}
a:visited{
    color: orange;
}

2. a:hover{
    color: blue;
} 

3. a:active{
    color: green;
}

4.input:focus{
    border: 1px solid red;
}
表示获得焦点时

七. 其他伪类选择器

1. a:not([href*="baidu"]){
    color: red;
}

否定选择器，反选

2. :empty{
    display: none;
}

匹配没有任何内容的元素

3. :lang(en){
    color: red;
}

选择包含lang属性，属性值前缀为en的元素。和属性选择器匹配结果一致。

4. :target{
    color: red;
}

定位到锚点时，选择此元素

5. ::selection{
    color: red;
}

这是一个伪元素选择器，当选择文字时触发选择。
