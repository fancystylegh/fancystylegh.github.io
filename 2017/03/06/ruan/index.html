今天准备一边听听李炎恢老师的js视频，一边看看阮大神的js参考教程，来好好研究一下。。
昨天看了一会儿切片，这么简单！上次面试一家小公司一听我没有切过就不要了。这个学起来明明很快啊。。

var a = 1 + 3;

对象分为三个子类型：对象、object、函数

因为typeof运算符对数组和对象显示结果都是object，可以用instanceof运算来区分

布尔值为假的情况：

  • undefined
  • null
  • false
  • 0
  • NaN
  • “”或’’(空字符串)

浮点数不是精确的值，运算要小心

isNaN

isFinite,检查某个值是不是正常数值，而不是infinity。

与数值相关的全局方法
parseInt()将字符串转成整数
parseInt(1.23);等同于 parseInt('1.23'); //1
如果parseInt的参数都是字符串，结果只返回字符串头部可以转成数字的部分；如果字符串的第一个字符不能转化成数字（后面跟着数字的正负号除外），返回NaN。

parseInt('0x10'); //16

进制转换，第二个参数只有在2-36之间才有意义，超出范围返回NaN，0、undefined和null直接忽略
parseInt('1000',8); //512转换成八进制

parseFloat()方法用于将一个字符串转换为浮点数
parseFloat([]); //NaN

由于字符串是只读的，只有将字符串显式转为数组才能使用。

Base64转码：这是一种编码方法，可以将任意字符转成可打印的字符。使用这个是为了不出现特殊字符，简化程序的处理。
btoa(): 字符串或二进制值转为Base64编码
atob(): Base64编码转为原来的编码

将非ASCII码字符转为Base64编码，必须中间插入一个转码环节。

对象的所有的键名都是字符串，所以加不加引号都可以。
对象的每一个键名又称为属性（property）,它的键值可以是任意数据类型，如果一个属性的值为函数，通常把这个属性成为方法，可以像函数那样调用。

对象的引用
不同的变量名指向同一个对象，那它们都是这个对象的引用，也就是说指向同一个内存地址。修改其中的一个变量，会影响到其他所有变量。

javascript规定，如果行首是大括号，一律解释为语句（即代码块）。如果要解释为表达式（即对象），必须在大括号前加上圆括号。({foo:123})

读取属性：一是使用点运算符，还有一种是使用方括号运算符

js允许属性的’后绑定’，可以在任何时刻新增属性。

查看一个对象本身的所有属性，可以使用Object.keys方法

删除对象的属性：delete o.p;

in运算符：用于检查对象是否包含某个属性，包含就返回true

可以用in运算符判断一个全局变量是否存在。不能识别继承的属性。
if('x' in window){return 1;}

for…in循环用来遍历一个对象的全部属性。不仅可以遍历对象自身的属性，还遍历继承的属性。

如果只想遍历对象本身的属性，可以使用hasOwnProperty方法，在循环内部判断是否是自身的属性。

将数组清空的一个有效方法，就是将length属性设为0。

类似数组的对象：明显的特征就是具有length属性。但是这里的length不会随着成员的变化而变化。
典型的类似数组的对象是arguments对象，以及大多数的DOM元素集，还有字符串。

数组的slice方法将类似数组的对象变成真正的数组
var arr = Array.prototype.slice.call(arrayLike);

遍历类似数组的对象，可以采用for循环，也可以采用数组的forEach方法

数组的遍历可以使用for循环或while循环

数组的forEach方法也可以用来遍历数组

length属性返回函数预期传入的参数个数，即函数定义之中的参数个数。

函数的toString方法返回函数的源码，函数内部的注释也可以返回

函数参数如果是原始类型的值（数值、字符串、布尔值），传递方式是传值传递，无论怎么修改都不会影响原始值；函数参数是复合类型的值（数组、对象、其他函数），传递方式是传址传递，在函数内部修改参数会影响原始值。

arguments对象包含了函数运行时的所有参数，可以在函数体内部读取所有参数。
slice方法和逐一填入新数组，将arguments转为真正的数组

arguments对象带有一个callee属性，返回它所对应的原函数。

闭包
js特有的“链式作用域”结构，子对象会一级一级的向上寻找所有父对象的变量。
只有函数内部的子函数才能读取内部变量，因此可以把闭包理解为“定义在函数内部的函数”。
闭包就是将函数内部和函数外部连接起来的一座桥梁

外层函数每次运行，都会产生一个新的闭包，而这个闭包又会保留外层函数的内部变量，会很耗内存。所以不能滥用闭包。

在js中，()是一种运算符，表示调用该函数。

不能让function出现在行首。让引擎将它认为是一个表达式。
(function(){...})(); //立即调用的函数表达式

通常情况下，只对匿名函数使用这种“立即执行的函数表达式”。
它的目的有两个：一是不必为函数命名，避免了污染全局变量；
二是IIFE内部形成了一个单独的作用域，可以封装一些外部无法读取的私有变量

eval命令：将字符串当作语句执行。

为了得到正确的负数余数值，需要先使用绝对值函数

对于两个对象的比较，严格相等运算符比较的是地址，而大于或小于运算符比较的是值

比较不同类型的数据时，相等运算符会先将数据进行类型转换，然后再用严格相等运算符比较。

不管什么类型的值，经过取反运算后，都变成了布尔值;如果对一个值连续做两次取反运算，等于将其转为对应的布尔值，与Boolean函数的作用相同

强制转换：主要指使用Number、String和Boolean三个构造函数，手动将各种类型值转换成数字、字符串或布尔值。

Number函数将字符串转换成数值很严格，只要有一个不能就转成NaN。
Number方法的参数是对象时，将返回NaN。

String方法的参数如果是对象，返回一个类型字符串；如果是数组，返回该数组的字符串形式

使用Boolean函数，可以将任意类型的变量转为布尔值，所有对象（包括空对象）的转换结果都是true

错误处理机制

  • SyntaxError是解析代码时发生的语法错误
  • ReferenceError是引用一个不存在的变量时发生的错误
  • RangeError是当一个值超出有效范围时发生的错误
  • TypeError是变量或参数不是预期类型时发生的错误
  • URIError是URI相关函数的参数不正确时抛出的错误

行尾的分号
不需要的情况：for和while循环（需要注意的是do…while循环是有分号的）
分支语句：if，switch，try
函数的声明语句（但是函数表达式仍然要使用分号）
除了这三种，其他的最好都加。

应该避免使用全局变量。如果不得不用，用大写的字母表示变量名。

建议使用Object.create()命令，替代new命令
如果不得不使用new，为了防止出错，最好在视觉上把构造函数与其他函数区分开来
构造函数的函数名，采用首字母大写
